# Méthode de visualisation : 

Une idée, une carte = une graine

2 idées, un segment, une tige

Plusieurs points, des lianes

En fonction du temps : évolutions d'une suite d'idées sur un même tag = une branche

Autre idée qui a germé beaucoup plus tard dans une même direction, nouvelle branche dans le même axe

Référence, à une note passée, une racine

Le temps fait évoluer le tronc comme les racines

Les tags sont les axes, et chaque graine finit par prendre forme dans cet arbre
